This plugin extends WooCommerce with the Viva Wallet payment gateway, allowing you to accept payments using Viva Wallet's secure payment processing platform.

== Description ==

Accept payments seamlessly through Viva Wallet's robust and secure platform, directly integrated into your WooCommerce store. This plugin provides a smooth checkout experience for your customers while offering you a reliable and efficient payment solution.

== Features ==

*   Securely process payments using Viva Wallet.
*   Redirect customers to the Viva Wallet platform for a seamless checkout experience.
*   Automatic order status updates via webhooks for efficient order management.
*   Transaction IDs are automatically saved to WooCommerce orders for easy tracking and reconciliation.
*   Customize the gateway's title and description to match your store's branding.
*   Support for both live and demo (testing) environments to ensure smooth integration.
*   Automatic updates: Stay up-to-date with the latest features and security enhancements directly through your WordPress dashboard.

== Installation ==

1.  **Upload the plugin:**  In your WordPress dashboard, go to **Plugins > Add New**. Click **Upload Plugin** and select the plugin's zip file.
2.  **Activate the plugin:** Go to **Plugins > Installed Plugins** and activate the "WooCommerce Gateway Viva Wallet" plugin.
3.  **Configure the plugin:** Go to **WooCommerce > Settings > Payments** and click on "Viva Wallet" to configure the gateway settings.

== Frequently Asked Questions ==

= How do I obtain my Viva Wallet API credentials? =
Log in to your Viva Wallet account and navigate to the API Keys or Developer section. You can generate your Client ID (Merchant ID) and Client Secret (API Key) there.

= What is the Live Source Code List? =
The Live Source Code List is a comma-separated list of source codes associated with your Viva Wallet account. You can find these codes in your Viva Wallet account settings.

= How do I set up webhooks? =
Refer to the Webhooks section in the full README file on GitHub for detailed instructions on setting up webhooks in your Viva Wallet account.
